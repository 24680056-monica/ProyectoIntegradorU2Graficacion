# ProyectoIntegradorU2Graficacion

Para el desarrollo de éste proyecto se ocupa inicialmente un boceto de poses, ya que será una animación por frames.

En este caso se utilizará la siguiente imagen como guía.
<img width="1264" height="842" alt="Gemini_Generated_Image_p6w3idp6w3idp6w3" src="https://github.com/user-attachments/assets/a8828479-d486-4cd5-899b-7323b7c6f7bf" />.

## Inicio
Primero se abre un documento en blender como 2D, una vez hecho eso se coloca en modo objeto y se presiona shift + a, para agregar una imagen como guía. En este caso se agregará la imagen que se mostró anteriormente. Después se coloca en la parte inferior izquierda línea de tiempo y se pondrá el límite de tiempo que tendrá nuestra animación.

# Dibujo
En el segundo 1, se dibuja la primer pose, después se contea haste el segundo 4 (nuestra animación avanzará cada tres segundos) y se dibuja la segunda pose. Se hará así sucesivamente hasta terminar de dibujar pose por pose.

## Final
Para comprobar como se ve la animación simplemente se le coloca play y la animación empezará en ciclo reproduciendo y así observandose la animación por frames.
